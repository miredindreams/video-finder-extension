# Video Finder

## Concept
 - User goes to a website (let's say kinopoisk), chooses a movie/series/anime that he wants to watch, but he is not satisfied with the quality or voice acting that is presented on the site. And at this point, my extension comes into play, which, using filters (voice acting, quality), will show the user on which site there is what he wants to watch, analyzing the information on the screen, or through a search filter string (by name). After finding a match using the filters, he will see a list with previews and links where he can watch in the quality and dubbing he wants.

### Features
- Automatic parsing of movie pages
- Search by quality and dubbing
- Support for multiple websites
- Caching of results

# Video Finder

Браузерное расширение для поиска альтернативных источников просмотра видео.

## Концепт: 
- Пользователь заходит на сайт (допустим кинопоиск), выбирает фильм/сериал/аниме, которое он хочет посмотреть, но его не устраивает качество или озвучка, которая представлено на сайте. И на этом моменте вступает в ход мое расширение, которое с помощью фильтров(озвучка,качество) покажет пользователю на каком сайте есть то, что он хочет посмотреть, анализируя информацию на экране, либо через строку-фильтр поиска(по названию). После нахождения совпадения по фильтрам, у него появится список с превью и ссылками, где он может посмотреть в нужном ему качестве и с нужной ему озвучкой.

### Возможности
- Автоматический парсинг страниц фильмов
- Поиск по качеству и озвучке
- Поддержка multiple сайтов
- Кэширование результатов
